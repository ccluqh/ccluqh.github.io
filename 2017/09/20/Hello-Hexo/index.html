macOS 安装Hexo

1.在git官网下载git

点击安装包按照提示顺序完成安装即可

2.安装Node.js

我第一次直接在官网下载安装包安装，结果最后配置失败，建议通过brew安装

/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

换行继续输入

brew install node

使用淘宝npm镜像进行包管理

npm install -g cnpm --registry=https://registry.npm.taobao.org

安装成功后，就可以能过以下指令加速安装相应包

cnpm install [packageName]

3.安装Hexo

通过node的npm包管理工具安装Hexo，执行以下指令(当然cnpm可以换成npm，这里使用淘宝加速)

cnpm install -g hexo

在本地创建管理自己博客的文件目录，cd进入目录，执行初始化代码

hexo init

安装依赖包

cnpm install

生成静态页面

hexo g

启动本地服务

hexo s

打开浏览器，输入 http://localhost:4000/ ，看到以下页面说明hexo环境安装成功
